### Getting the raw link of `image` file from the github

GitHubâ€™s UI no longer shows a **Raw** button for images. To copy the direct/raw link:

1. Open the image file (example: `https://github.com/ettechlimited/cdn/blob/main/et.png`)
2. Right-click on the image.
3. Select **Copy Image Address**.

The copied URL should end with `?raw=true`, for example:  
`https://github.com/ettechlimited/cdn/blob/main/et.png?raw=true`
